### 2. 打包构建脚本

**scripts/package.js** (6.8 KB)

功能流程:
1. 验证 skill.yaml 完整性
2. 清理输出目录
3. 编译 TypeScript (`npx tsc`) - 容错模式，即使类型错误也继续
4. 复制编译后的 dist/* 文件到输出目录
5. 复制资源文件 (docs, examples, README, LICENSE)
6. 生成 `manifest.json` (包含构建元数据: 源仓库、commit hash)
7. 验证打包结果完整性

使用示例:
```bash
node scripts/package.js                    # 默认输出到 dist/skill
node scripts/package.js --output-dir release  # 自定义输出目录
node scripts/package.js --skip-build       # 跳过编译（用于重新打包）
```
