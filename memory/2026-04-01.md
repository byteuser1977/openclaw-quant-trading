### 🔧 Worker Isolation 调试历程

1. **路径混乱**: Jest 中的 `__dirname` 导致模块解析错误
   - 解决: 使用 `../../../` 计算项目根，传递绝对路径
2. **模块依赖**: 原始 TS Worker 依赖未编译的模板模块
   - 解决: 创建自包含的 JS 版本，内联策略映射
3. **参数解析**: Worker 不支持 `--strategy value` 格式
   - 解决: 同时支持 `--strategy=value` 和 `--strategy value`
4. **输出缓冲**: 子进程 `process.exit(0)` 过早， stdout 未刷新
   - 解决: 使用 `setImmediate()` 延迟退出，确保 I/O 完成
